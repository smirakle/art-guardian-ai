## Technical Production Notes

### Video Requirements:
- **Resolution**: 4K (3840x2160) for crisp detail
- **Frame Rate**: 30fps for smooth demonstration
- **Aspect Ratio**: 16:9 for platform compatibility
- **Duration**: 15-18 minutes (can be edited to shorter versions)

### Screen Recording Setup:
1. Use high-resolution displays (minimum 1920x1080)
2. Enable cursor highlighting and click animations
3. Prepare sample content and test data beforehand
4. Use professional screen recording software (OBS, Camtasia)
5. Record in segments for easier editing

### Audio Production:
- **Professional voiceover** with clear articulation
- **Background music**: Subtle, technology-focused (royalty-free)
- **Sound effects**: Minimal, for transitions and notifications
- **Audio levels**: Voiceover primary (-12dB), music ambient (-24dB)

### Visual Elements to Highlight:
- TSMO branding consistency throughout
- Real-time processing animations
- Threat level color coding (red/orange/green)
- Professional report formatting
- Blockchain verification graphics
- API integration examples

### Content Preparation:
- Sample documents (PDF, images, artwork)
- Test user accounts with various subscription levels
- Pre-configured detection results
- Sample licensing agreements
- Mock payment transactions
- Prepared legal documents

### Editing Guidelines:
- Smooth transitions between sections
- Callout graphics for important features
- Progress bars and loading animations
- Professional lower thirds for feature names
- Consistent pacing (not too fast, not too slow)
- Clear section breaks with title cards

### Distribution Formats:
- **Full Version**: 15-18 minutes comprehensive demo
- **Executive Summary**: 5-7 minutes key features only
- **Feature Spotlights**: 2-3 minute individual feature demos
- **Social Media Clips**: 30-60 second highlights

This comprehensive demo script showcases every major TSMO feature while maintaining a logical flow that demonstrates real-world usage patterns and value proposition.
